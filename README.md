# Multiplayer 3D Game (Node.js + Three.js + Socket.IO)

## 🎮 Description
Real-time multiplayer 3D game with rooms, matchmaking and progression system.

## 🚀 Tech Stack
- Node.js
- Express
- Socket.IO
- MongoDB + Mongoose
- Three.js
- JWT Authentication

## ⚙️ Features
- Real-time multiplayer
- Rooms system
- Matchmaking
- Player progression (currency, stats)
- Shop system
- 3D client (Three.js)
- Server-authoritative movement

## 🧪 Run locally

```bash
npm install
npm run dev
